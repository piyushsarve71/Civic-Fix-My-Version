# Civic-Fix 🚀

A real-time civic infrastructure reporting platform designed to connect citizens with local authorities for faster resolution of issues.

## 💡 About this Project
This project is an independent implementation of the Civic-Fix platform. I have structured this to focus on scalable UI components and streamlined data handling.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express
- **Database:** MongoDB

## 🚀 Key Features
- **User-Centric Reporting:** Intuitive form for reporting issues with location data.
- **Real-time Status Tracking:** Users can see the current progress of their reported issues.
- **Admin Dashboard:** A dedicated panel for authorities to prioritize and manage reports.

## ⚙️ How to Run Locally
1. Clone the repository: `git clone https://github.com/piyushsarve71/Civic-Fix-My-Version.git`
2. Install dependencies: `npm install`
3. Start the server: `npm start`
